# ESP32 Function Generator

## Project Structure
- app/        : Application logic and state machine
- signal/     : Waveform generation and frequency control
- input/      : Buttons, encoder, user input handling
- display/    : OLED/LCD display and waveform preview
- drivers/    : Hardware-specific drivers (DAC, timers)
- utils/      : Debug and helper utilities

## Team Rules
- Each member owns one folder
- Do not work directly on main
- All changes via branches and pull requests
